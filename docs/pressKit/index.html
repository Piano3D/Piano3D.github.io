DESCRIPTION

Piano 3D is a realistic mobile piano app made with Unity 3D. Apart from various sounds, authentic keys animations and appearance customization, it also features simplified tutorials which can help users learn their favorite songs.

FEATURES

  • Full size, 88-keys keyboard
  • Scrolling and scaling
  • Realistic keys animation
  • 2D/3D mode
  • Various sounds
  • ADSR filter
  • 60+ tutorials
